Interface IAutoBeRealizeControllerProps

Configuration properties for generating NestJS controller implementations.

This interface defines the essential inputs required for the Realize compiler's controller generation process, providing specifications for API endpoints, business logic functions, and authorization mechanisms. The properties enable the compiler to create fully-integrated controller classes that properly orchestrate business logic execution with appropriate security controls.
